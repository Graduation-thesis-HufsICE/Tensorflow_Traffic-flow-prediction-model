# Tensorflow_Traffic-flow-prediction-model

#### Tensorflow 와 Keras를 활용한 교통 흐름 예측 모델
1. 서울시 공공데이터를 통한 학습 데이터 전처리
2. 표준화를 통한 2가지 특성(요일 값, 시간 값) 활용
3. LSTM Layer를 통한 시계열 예측 모델 생성
